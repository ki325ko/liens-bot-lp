LIENS BOT 利用規約

制定日: 2026年3月26日

LIENS（以下「当方」）が提供する「LIENS BOT」（以下「本サービス」）の利用に関して、当方は本利用規約（以下「本規約」）を以下のとおり定めます。

第1条（適用）

  1. 本規約は、当方が提供する本サービスの利用に関し、当方と利用者との間に適用される基本的な事項を規定します。
  2. 「利用者」とは、本規約に同意のうえ本サービスの利用を申し込み、当方がこれを承諾した者をいいます。
  3. 利用者は、本サービスの利用開始にあたり、本規約に同意するものとします。本サービスのウェブサイト上で「利用規約に同意する」にチェックを入れた時点で、本規約に同意したものとみなします。
  4. 本規約のほか、当方が別途定めるプライバシーポリシー、ガイドライン等も本規約の一部を構成するものとします。
  5. 本規約の規定と個別規定の規定が矛盾する場合には、個別規定において特段の定めなき限り、個別規定の規定が優先されるものとします。

第2条（サービスの概要）

当方が提供する本サービスの概要は以下のとおりとします。

  1. ウェブサイト等に設置するチャットボットの提供
  2. チャットボットのシナリオ設計・設定支援
  3. 生成AIを活用した自動応答機能の提供
  4. チャットボットの利用状況に関するデータ分析・レポート
  5. 上記に関連する付帯サービス

第3条（無料トライアル）

  1. 当方は、利用者に対し、本サービスの申込日から14日間の無料トライアル期間（以下「トライアル期間」）を提供します。
  2. トライアル期間中、利用者は本サービスの機能を無料で利用することができます。ただし、当方の判断により、一部の機能を制限する場合があります。
  3. トライアル期間の終了後、利用者が有料プランへの申込みを行わない場合、本サービスの利用は停止されます。トライアル期間の終了をもって、有料プランへ自動的に移行することはありません。
  4. トライアル期間中であっても、利用者は本規約の全条項を遵守するものとします。

第4条（有料プランおよび料金）

  1. 利用者がトライアル期間終了後も本サービスの利用を希望する場合、当方所定の方法により有料プランの申込みを行うものとします。
  2. 本サービスの有料プランの種類および月額利用料金は、以下のとおりとします。
プラン名1年契約6ヶ月契約3ヶ月契約
ハイブリッドプラン50,000円/月60,000円/月80,000円/月
生成AIプラン30,000円/月36,000円/月48,000円/月
シナリオプラン30,000円/月36,000円/月48,000円/月
  1. 上記の金額はいずれも税別とします。利用者は、利用料金に加え、消費税相当額を負担するものとします。
  2. 当方は、利用者に対して初期費用を請求する場合があります。初期費用の金額は、申込時に当方が提示するものとし、利用者が申込みを完了した時点で合意されたものとみなします。初期費用の定めがない場合は無料とします。
  3. 当方は、相当期間前に利用者に通知することにより、利用料金を改定することができるものとします。

第5条（支払方法）

  1. 利用者は、利用料金を当方が指定する方法（クレジットカード決済、銀行振込等）により支払うものとします。
  2. 銀行振込の場合、振込手数料は利用者の負担とします。
  3. 初期費用が発生する場合、利用者は有料プランの利用開始日までに初期費用を支払うものとします。
  4. 利用者が利用料金の支払を遅延した場合、利用者は当方に対し、年14.6%の割合（年365日の日割計算）による遅延損害金を支払うものとします。

第6条（契約期間および自動更新）

  1. 有料プランの契約期間は、利用者が選択した期間（1年、6ヶ月、または3ヶ月）とします。
  2. 契約満了日の1ヶ月前までに、管理画面上またはその他当方所定の方法で利用者または当方から更新拒絶の意思表示がない場合、本契約は同一条件で自動更新されるものとし、以後も同様とします。

第7条（中途解約および返金）

  1. 利用者は、管理画面から解約手続きを行うことができます。解約した場合、契約期間終了まで引き続き本サービスを利用できます。
  2. 理由のいかんを問わず、既に支払われた利用料金および初期費用の返金は行いません。

第8条（アカウントの管理）

  1. 利用者は、自己の責任において、本サービスのアカウント情報（ID・パスワード等）を適切に管理・保管するものとし、これを第三者に利用させ、または貸与・譲渡・売買等をしてはならないものとします。
  2. 利用者は、正当な権限を有する役員および従業員以外にアカウント情報を利用させてはならないものとします。
  3. アカウント情報の管理不十分、使用上の過誤、第三者の使用等による損害の責任は利用者が負うものとし、当方は一切の責任を負いません。
  4. 利用者は、アカウント情報が第三者に使用されていることが判明した場合、直ちに当方に通知し、当方の指示に従うものとします。

第9条（登録情報の変更）

  1. 利用者は、登録した情報に変更があった場合、速やかに当方所定の手続きにより届け出るものとします。この届出がない場合、当方は登録情報の変更がないものとして取り扱うことができるものとします。
  2. 利用者が前項に基づく届出を怠ったことにより利用者が不利益を被ったとしても、当方はその責任を負いません。

第10条（禁止事項）

利用者は、本サービスの利用にあたり、以下の行為をしてはなりません。

  • 法令または公序良俗に違反する行為
  • 当方または第三者の知的財産権、肖像権、プライバシー、名誉その他の権利を侵害する行為
  • 本サービスのネットワークまたはシステムに過度な負荷をかける行為
  • 本サービスに不正にアクセスし、または不正なアクセスを試みる行為
  • 逆アセンブル、逆コンパイル、リバースエンジニアリング等によって本サービスのソースコードを解析する行為
  • 本サービスのウェブサイトやソフトウェアを複製、送信、譲渡、貸与または改変する行為
  • 本サービスを通じて、コンピュータウイルスその他の有害なプログラムを送信する行為
  • 他の利用者または第三者になりすます行為
  • 本サービス上のアカウントまたはコンテンツを第三者に有償で貸与、譲渡、売買等をする行為
  • 反社会的勢力等への利益供与その他の協力行為
  • 本サービスの運営を妨害するおそれのある行為
  • その他、当方が不適切と判断する行為

第11条（本サービスの変更・中断・終了）

  1. 当方は、利用者に事前に通知することなく、本サービスの内容を変更し、または追加することができるものとします。
  2. 当方は、以下のいずれかに該当する場合、利用者への事前の通知なく、本サービスの全部または一部を一時的に中断することができるものとします。
    • システムの保守・点検・更新を行う場合
    • 天災、火災、停電その他の不可抗力により本サービスの提供が困難な場合
    • システムに予期せぬ障害が発生した場合
    • その他、当方が中断を必要と判断した場合
  3. 当方は、当方の判断により本サービスの全部または一部を終了することができるものとします。この場合、当方は合理的な期間をもって利用者に事前に通知するよう努めます。
  4. 本サービスの全部の廃止に伴い利用者がすでに支払った利用料金については、廃止日をもって日割計算し、残余の期間に対応する額を返金します。

第12条（知的財産権）

  1. 本サービスに関する知的財産権は、すべて当方または当方にライセンスを許諾している者に帰属します。
  2. 本規約に基づく本サービスの利用許諾は、当方の知的財産権の譲渡または使用許諾を意味するものではありません。
  3. 利用者が本サービスに登録・入力したデータの権利は、利用者に帰属します。ただし、当方は本サービスの提供および改善のために当該データを利用できるものとします。

第13条（データの取扱い・保持）

  1. 利用者が本サービスに登録・入力したデータ（以下「利用者データ」）の権利は利用者に帰属します。
  2. 当方は、本サービスの提供および改善の目的で、利用者データを利用できるものとします。ただし、第三者に対して利用者を特定できる形で開示することはありません。
  3. 当方は、利用者データを契約終了後90日間保持するよう努めます。ただし、利用者データの保存を保証するものではありません。
  4. 利用者は、契約終了後は利用者データにアクセスできなくなることをあらかじめ了解するものとします。データのエクスポートを希望する場合、契約終了日までに当方に申し出るものとします。
  5. 当方は、利用者データを第三者に一切開示せず、本サービスの遂行以外のいかなる目的にも使用しません。ただし、以下の場合を除きます。
    • 利用者の同意がある場合または法令に基づく場合
    • 運用データを個人が特定できないように匿名化し、サービス改善や統計分析に利用する場合
    • 障害の解消またはセキュリティの確保等、本サービスの運用に必要と当方が判断した場合

第14条（個人情報の取扱い）

  1. 当方は、利用者の個人情報を、別途定めるプライバシーポリシーに従い、適切に取り扱います。
  2. 利用者は、本サービスの利用に関して取得したエンドユーザーの個人情報について、自らの責任において適用法令に従い適切に管理するものとします。

第15条（秘密保持）

  1. 当方および利用者は、本サービスの利用に関連して知り得た相手方の非公開情報（以下「秘密情報」）を、相手方の事前の書面による承諾なく、第三者に開示または漏洩してはなりません。ただし、以下の情報は秘密情報に含まないものとします。
    • 開示を受けたときに既に保有していた情報または既に公知であった情報
    • 開示を受けた後、秘密保持義務を負うことなく第三者から正当に入手した情報
    • 開示を受けた後、相手方から開示を受けた情報に関係なく独自に取得し、または創出した情報
    • 開示を受けた後、自己の責めに帰し得ない事由により公知となった情報
  2. 前項の秘密保持義務は、本サービスの利用終了後もなお存続するものとします。

第16条（サービスレベル）

  1. 当方は、本サービスの月間稼働率99%を目標とします。ただし、これは努力目標であり、保証するものではありません。
  2. 計画メンテナンスによるサービス停止は、原則として事前に利用者に通知します。

第17条（免責・非保証）

  1. 当方は、本サービスの内容について、その完全性、正確性、有用性、安全性、信頼性、特定目的への適合性等について、いかなる保証も行いません。
  2. 当方は、本サービスに中断、中止その他の障害が生じないことを保証しません。
  3. 当方は、本サービスの利用により利用者に生じた損害について、当方に故意または重大な過失がある場合を除き、一切の責任を負いません。
  4. 何らかの外的要因（システムの故障や外部からの攻撃等）により、本サービス内のデータが破損した場合は、当方はそれらのデータの復活の責任および損害賠償の責任を負わないものとします。
  5. 本サービスの利用に必要な機器、通信環境等はすべて利用者の費用と責任で準備するものとします。

第18条（損害賠償）

  1. 当方が利用者に対して損害賠償責任を負う場合であっても、当方の賠償額は、利用者が当方に対して過去12ヶ月間に支払った利用料金の総額を上限とします。なお、トライアル期間中の場合は、当方の損害賠償責任は負わないものとします。
  2. 利用者が本規約に違反し、当方に損害を与えた場合、利用者はその全額（弁護士費用を含みます）を賠償するものとします。

第19条（利用の停止・解除）

  1. 当方は、利用者が以下のいずれかに該当した場合、事前の通知なく、本サービスの利用を停止し、または利用契約を解除することができます。
    • 本規約に違反したとき
    • 登録情報に虚偽があることが判明したとき
    • 支払停止もしくは支払不能の状態に陥ったとき、または破産手続等の開始の申立てがあったとき
    • 当方からの連絡に対して14日間以上応答がないとき
    • 反社会的勢力等に該当すると当方が判断したとき
    • その他、当方が利用の継続を不適当と判断したとき
  2. 前項による解除の場合、利用者は期限の利益を喪失し、直ちに当方に対する一切の債務を履行するものとします。

第20条（実績表記の許諾）

利用者は、当方がウェブサイト、営業資料等において、利用者が本サービスを利用している旨を実績として公表すること、およびその際に利用者の名称・ロゴ等を使用することを許諾するものとします。ただし、利用者が書面により不同意を申し出た場合はこの限りではありません。

第21条（反社会的勢力の排除）

利用者は、自己または自己の役員、従業員等が、暴力団、暴力団員、暴力団関係企業、その他の反社会的勢力等に該当しないこと、および将来にわたって該当しないことを表明・保証するものとします。当方は、利用者がこれに違反した場合、何らの催告を要することなく利用契約を解除できるものとします。

第22条（事業譲渡）

当方は本サービスにかかる事業を他社に事業譲渡（事業譲渡、会社分割その他事業が移転するあらゆる場合を含みます）した場合には、当該事業譲渡に伴い利用契約上の地位、本規約に基づく権利および義務ならびに利用者の情報を当該事業譲渡の譲受人に譲渡することができるものとします。利用者は、かかる譲渡につき予め同意したものとみなします。

第23条（規約の変更）

  1. 当方は、以下の各号のいずれかに該当する場合、利用者の個別の承諾を得ることなく、本規約を変更できるものとします。
    • 変更が利用者の一般の利益に適合する場合
    • 変更が契約の目的に反せず、かつ合理的なものである場合
  2. 当方は、本規約を変更する場合、変更内容および効力発生時期を、本サービスのウェブサイトへの掲載その他適当な方法により、事前に利用者に通知します。

第24条（連絡・通知）

本サービスに関する当方から利用者への連絡・通知は、電子メール、本サービスのウェブサイト上の掲示その他当方が適当と判断する方法により行うものとします。当方は、利用者から変更届け出がない限り、現在登録されている連絡先が有効なものとみなして通知を行い、発信時に利用者へ到達したものとみなします。

第25条（権利義務の譲渡禁止）

利用者は、当方の書面による事前の承諾なく、本規約上の地位または本規約に基づく権利義務の全部もしくは一部を、第三者に譲渡し、または担保に供してはなりません。

第26条（分離可能性）

本規約のいずれかの条項またはその一部が法令等により無効または執行不能と判断された場合であっても、本規約の残りの規定は、継続して完全に効力を有するものとします。

第27条（準拠法および管轄裁判所）

  1. 本規約の準拠法は日本法とします。
  2. 本規約に起因しまたは関連する一切の紛争については、福井地方裁判所を第一審の専属的合意管轄裁判所とします。

第28条（協議解決）

当方および利用者は、本規約に定めのない事項または本規約の解釈に疑義が生じた場合には、互いに誠意をもって協議のうえ速やかに解決を図るものとします。
